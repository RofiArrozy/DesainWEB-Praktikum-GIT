Testimoni Desain Web

Mata kuliah Desain Web membuka wawasan saya tentang bagaimana tampilan sebuah situs dibangun dari nol. Melalui HTML dan CSS, saya belajar menyusun struktur halaman dengan rapi sekaligus mengatur tampilannya agar menarik. Awalnya terlihat rumit, tapi semakin dipelajari semakin terasa menyenangkan, terutama saat melihat hasil kode sederhana berubah menjadi desain nyata di layar. Dan terasa puas jika menemukan kesalahan di dalam kode yang dibuat.

KRITIK

Materi dan penjelasan terlalu cepat untuk baru yang mempelajari HTML dan CSS

SARAN
Mungkin akan sangat membantu jika di setiap akhir penjelasan konsep besar (seperti Flexbox atau Grid), kami diberi waktu 5-10 menit untuk langsung mencoba sendiri atau diberi studi kasus kecil untuk dipecahkan.
